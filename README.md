## About TALL Dashboard (UNDER ACTIVE DEVELOPMENT)

TALL Dashboard is a dashboard made with:

- Tailwind CSS.
- AlpineJS.
- Laravel.
- Livewire.

TALL Dashboard is still *under development*

## Installation
- clone the repository
- composer install/update
- cp .env.example .env
- php artisan key:generate
- npm install
- npm run dev
- php artisan serve
